# Session Briefing Template

Max length: 300 words

## Required sections
1. Urgent Mail Summary
2. Today's 3 MITs
3. Actionable Ops Note

## Rules
- Be concrete, not generic.
- Highlight approval-needed items clearly.
- Follow priority hierarchy: Revenue Blocker -> Bold Business -> W3J Product.
- If upstream state files are empty, say so explicitly instead of inferring.
