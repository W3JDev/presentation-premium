🔷 LangGraph Core Concepts

State

Shared mutable data across nodes

Nodes

Python functions that process state

Edges

Conditional routing between nodes

Key Insight: LangGraph treats agent workflows as directed graphs with cycles — enabling true agentic behavior.

💻 Minimal Agent Example

from langgraph.graph import StateGraph, END

# Define state schema
class AgentState(TypedDict):
    messages: list
    next_action: str

# Create graph
graph = StateGraph(AgentState)

# Add nodes
graph.add_node("think", think_node)
graph.add_node("act", act_node)

# Define edges
graph.set_entry_point("think")
graph.add_conditional_edges("think", should_continue)
graph.add_edge("act", "think")
graph.add_edge("act", END)

# Compile and run
app = graph.compile()
result = app.invoke({"messages": [...]})

🔗 MCP (Model Context Protocol)

What is MCP?

Open standard for AI ↔ tools communication

ComponentRole
ClientLLM application
ServerTool provider
ProtocolJSON-RPC 2.0

Standardized Tools

  • File system access
  • Database queries
  • API integrations
  • Code execution

🔄 ReAct Pattern

Reason → Act → Observe cycle

Thought: "I need to check the user's calendar"
Action: call_calendar_tool
Observation: "User has meeting at 3pm"
Thought: "Now I can suggest a time"
Why it works: Explicit reasoning traces → better debuggability → reliable agents
